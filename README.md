# Nolan's Portfolio Website
This is my personal portfolio website project which I built to deepen my understanding of web languages and to showcase my projects and personal goals!

## Tech Stack:

- Next.js 14 – App Router
 
- Node.js 24.5 – Javascript engine
  
- Tailwind CSS v4 – Styling
 
- next-themes – Dark mode support
 
- Linode – Deployment

 ## Getting Started:

  	git clone https://github.com/nolantuttle/portfolio-site.git # Clone the repo
  
  	cd portfolio-site # Navigate to root directory
  
  	npm install # Install all required packages/dependencies
  
  	npm run dev # This will run a development server on localhost:3000

    npm run build # This will build the site fully

    npm run start # Launch the site on your machine
